# Node- Neo4j Service Template 

This is a service for Neo4j database using NodeJS.
**[node-neo4j][]** library has been used (avialble on npm as 'neo4j')

This app implements basic GET and POST methods.
Currently it has no UI, as main intention is to create a template for the service only.

It has implemented the followings 

- CORS filter
- Winston logger
- Forever (to keep running the application after you log out, You need to install forever first, check npm repo)



[node-neo4j]: https://github.com/thingdom/node-neo4j
